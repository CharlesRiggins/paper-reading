# DeepSeek-V3.2: Pushing the Frontier of Open Large Language Models

arXiv: 2512.02556 | DeepSeek-AI (2025)
Code: [https://huggingface.co/deepseek-ai/DeepSeek-V3.2-Exp](https://huggingface.co/deepseek-ai/DeepSeek-V3.2-Exp/tree/main/inference)

## Files

- `s00_s01_frontmatter_intro.md` — Title, abstract, and introduction. Identifies three critical deficiencies in open-source models (architecture, resource allocation, agent capabilities) and introduces DeepSeek-V3.2's solutions: DSA for efficient long-context attention, a scalable RL framework with post-training budget exceeding 10% of pre-training cost, and a large-scale agentic task synthesis pipeline generating 1,800+ environments and 85,000+ prompts.
- `s02_architecture.md` — The DSA architecture section. Describes the prototype (lightning indexer + fine-grained token selection with top-k), its instantiation under MLA's MQA mode, and the two-stage continued pre-training: a dense warm-up stage (2.1B tokens, KL-divergence loss to align indexer with main attention) and a sparse training stage (943.7B tokens, 2048 top-k tokens, indexer detached from the main computational graph). Also covers parity evaluation showing no performance degradation and inference cost analysis demonstrating $\mathcal{O}(Lk)$ complexity.
- `s03_posttraining.md` — Post-training pipeline including specialist distillation across six domains and mixed RL training with GRPO. Details four key GRPO scaling techniques: unbiased KL estimate (correcting the K3 estimator), off-policy sequence masking (masking negative sequences with large policy divergence), keep routing (preserving expert routing paths for MoE stability), and keep sampling mask (maintaining identical action subspaces). Also covers thinking-in-tool-use with context management, cold-start prompting, and large-scale agentic tasks (24K+ code, 50K+ search, 4K+ general, 6K+ code interpreter environments).
- `s04_evaluation.md` — Comprehensive evaluation results across reasoning, coding, math, code agent, search agent, and tool-use benchmarks. DeepSeek-V3.2 matches GPT-5-High on reasoning tasks (AIME 93.1, HMMT 92.5), outperforms all open-source models on code agents (SWE Verified 73.1), and substantially narrows the tool-use gap vs. closed-source models. DeepSeek-V3.2-Speciale surpasses Gemini-3.0-Pro across multiple benchmarks and achieves gold medals in IMO 2025 (35/42), IOI 2025 (492/600), CMO 2025, and ICPC WF 2025 (10/12 problems). Also covers synthetic task ablation and context management strategies for search agents (Discard-all achieves 67.6 on BrowseComp).
- `s05_conclusion.md` — Summary and three acknowledged limitations: world knowledge breadth (fewer training FLOPs), token efficiency (requires more tokens than Gemini-3.0-Pro), and complex task solving capability.
- `s06_references.md` — Full reference list (32 entries) including prior DeepSeek works, competitor models (GPT-5, Gemini, Claude, Kimi, MiniMax), benchmarks, and methodologies (GRPO, PPO, MLA, MQA).
- `s07_appendix.md` — Appendices A–E: MLA MHA/MQA modes, cold start prompt templates, non-thinking agentic evaluation results, competition evaluation protocols (IOI, ICPC, IMO, CMO), and author list.
